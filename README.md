# onecomme_testrendaer
わんコメへのテストコメントを連打するから`renda`+`er`

## Overview
わんコメにテストコメントを連投する小ツールです  
ローカルにhtmlをダウンロードしてお使いください

## Usage
1. htmlをダウンロード
2. ブラウザでhtmlを実行
3. `単発送信` か `連続送信`
